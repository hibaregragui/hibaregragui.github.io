# Portfolio - Hiba Regragui

**Module :** Programmation Web 2 | **Niveau :** Licence L2 | **Type :** Projet individuel

---

## 📌 Description

CV personnel interactif développé avec HTML/CSS, jQuery et ReactJS. Ce site représente mes compétences techniques et personnelles acquises au cours de mon parcours académique.

---

## 🚀 Technologies utilisées

- **HTML5/CSS3** - Structure et design responsive
- **jQuery** - Animations, validation formulaire, timeline interactive
- **ReactJS** - Composants ProjectCard réutilisables
- **Chart.js** - Diagramme circulaire des compétences techniques
- **GitHub Pages** - Hébergement

---

## ✨ Fonctionnalités

- En-tête avec photo de profile, nom, titre et coordonnées (email, LinkedIn, GitHub)
- Section "À propos" avec biographie
- Compétences techniques (diagramme circulaire interactif)
- Compétences personnelles (barres de progression jQuery)
- Timeline de formation interactive (jQuery)
- Projets avec composants React (ProjectCard) 
- Formulaire de contact avec validation (jQuery)
- Design responsive (mobile/tablette/desktop)
- Animations au scroll
